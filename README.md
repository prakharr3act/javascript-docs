# JavaScript Black & White Docs

A modern, responsive JavaScript documentation website inspired by **MDN Docs** and **W3Schools**.  
Built using only **HTML and CSS**, with a clean **black & white theme** and colorful syntax-highlighted code blocks.

---

##  Features

- Fully responsive layout (desktop, tablet, mobile)
- Fixed sidebar navigation like MDN
- Working anchor links for all sections
- Large typography and spacious layout
- Black & white UI with colorful code syntax
- Clean and readable documentation structure
- No JavaScript, no frameworks

---

##  Tech Stack

- HTML5  
- CSS3  
- Pure CSS syntax highlighting  

---

##  Sections Included

- Introduction
- Syntax
- Variables
- Data Types
- Operators
- Conditions
- Loops
- Functions
- Arrow Functions
- Arrays
- Objects
- DOM
- Events
- Async JavaScript
- Best Practices

##  Getting Started

1. Clone the repository  
2. Open `index.html` in your browser  
3. Start reading or customizing the documentation
